## Global verbs

### `quell dashboard`

Launch the read-only web dashboard on the local machine.

| Flag | Type | Default | Description |
|---|---|---|---|
| `--port` / `-p` | integer | `7777` | Port to bind the dashboard on. |
| `--host` | text | `127.0.0.1` | Interface to bind to (default localhost). |
| `--no-open` | boolean |  | Do not auto-open the browser. |
| `--quiet` / `-q` | boolean |  | Suppress the interrupted notice. |
| `--no-color` | boolean |  | Disable ANSI colors in the wrapper notice. |
| `--help` | boolean |  | Show this message and exit. |

**Examples:**

```
quell dashboard                          # http://localhost:7777
quell dashboard --port 8080 --no-open    # custom port, no browser
quell dashboard --host 0.0.0.0           # bind on every interface
```

### `quell init`

Configure Quell for a project.

| Flag | Type | Default | Description |
|---|---|---|---|
| `--path` / `-p` | path |  | Project directory to configure (defaults to current directory). |
| `--yes` / `-y` | boolean |  | Run non-interactively from flags + $QUELL_* env vars (no prompts). |
| `--monitor` | text |  | Monitor type: local-file, http-poll, vercel, sentry. (--yes only) |
| `--log-path` | text |  | Log path for --monitor local-file. |
| `--http-url` | text |  | Health URL for --monitor http-poll. |
| `--vercel-project-id` | text |  | Project ID for --monitor vercel (prj_...). |
| `--sentry-org` | text |  | Org slug for --monitor sentry. |
| `--sentry-project` | text |  | Project slug for --monitor sentry. |
| `--notifier` | text |  | Notifier: discord, slack, telegram, or none. (--yes only) |
| `--telegram-chat-id` | text |  | Telegram chat ID (required for --notifier telegram). |
| `--llm-provider` | text |  | LLM provider: anthropic, openai, google, or ollama. |
| `--llm-model` | text |  | Full LiteLLM model string (overrides provider default). |
| `--quiet` / `-q` | boolean |  | Errors only. |
| `--no-color` | boolean |  | Disable ANSI colors. |
| `--help` | boolean |  | Show this message and exit. |

**Examples:**

```
quell init                                              # interactive
quell init --yes                                        # all defaults
quell init --yes --monitor local-file --log-path /var/log/app.log \
--llm-provider anthropic
QUELL_ANTHROPIC_API_KEY=sk-... quell init --yes
```
